# 不存在 VRF 时应如何订购 Exchange
{: #how-to-order-exchange-when-there-is-no-vrf-present}

对于不存在 VRF 的帐户，订购表单中不会显示**选择 VRF** 字段。在这种情况下，下订单之后会生成 {{site.data.keyword.cloud}} 支持凭单。在您下订单之后，凭单编号和详细信息会显示在 UI 屏幕中。
