# VRF가 없는 경우 Exchange 주문 방법
{: #how-to-order-exchange-when-there-is-no-vrf-present}

VRF가 없는 계정의 경우 **VRF 선택** 필드는 주문서에 표시되지 않습니다. 이 경우 주문 후에 {{site.data.keyword.cloud}} 지원 티켓이 생성됩니다. 주문 후 티켓 번호 및 세부사항이 UI 화면에 표시됩니다.
